Bienvenue dans Transform: Labyrinth, un jeu qui peut vous servir à comprendre le fonctionnement de la propriété "transform:" en l'utilisant pour aider le pion à sortir du labyrinthe.
Pour normaliser les déplacements l'unité de mesure à utiliser de préfèrence est le rem.

Pour ce niveau il faudra utiliser l'une des valeurs suivantes:

  • translateX(): effectue une translation sur l'axe horizontal.
  • translate(X, Y): effectue une translation sur l'axe horizontal ou sur les deux axes.

Par exemple: "transform: translateX(1rem);" déplacera le pion d'une case vers la droite.

Utilisez à nouveau la propriété "transform:" pour aider le pion à trouver la sortie. Souvenez-vous que l'unité de mesure à utiliser est le rem.

Pour ce niveau il faudra encore utiliser l'une des valeurs suivantes:

  • translateX(): effectue une translation sur l'axe horizontal.
  • translate(X, Y): effectue une translation sur l'axe horizontal ou sur les deux axes.

La propriété "transform:" permet également de bouger verticalement. Souvenez-vous que l'unité de mesure à utiliser est le rem.

Pour s'échapper de ce niveau il faudra utiliser l'une des valeurs suivantes:

  • translateY(): effectue une translation sur l'axe horizontal.
  • translate(X, Y): effectue une translation sur l'axe horizontal ou sur les deux axes.

Par exemple: "transform: translateY(1rem);" déplacera le pion d'une case vers le bas.

Toujours en utilisant "transform:".

Echappez-vous en utilisant l'une des valeurs suivantes:

  • translateY(): effectue une translation sur l'axe horizontal.
  • translate(X, Y): effectue une translation sur l'axe horizontal ou sur les deux axes.

Rappel: l'unité de mesure à utiliser est le rem.

C'est le moment de combiner les déplacements "transform:".

Le seul moyen de sortir de cet étage est de combiner les déplacements horizontaux et verticaux, pour ce faire vous pouvez utiliser la valeur suivante:

  • translate(X, Y): effectue une translation sur l'axe horizontal ou sur les deux axes.

Rappel: l'unité de mesure à utiliser est le rem.

Changeons un peu les règles à notre avantage grâce à "transform:" et ses rotations.

Plusieurs options s'offrent à vous pour ce niveau:

  • rotateZ(): effectue une rotation sur la profondeur
  • rotateY(): effectue une rotation sur l'axe vertical'

Rappel: l'unité de mesure à utiliser pour les angles est: deg.

Oups, il semble que la sortie soit trop petite pour pouvoir s'échapper !

Essayez de lui rendre sa taille normale en utilisant la propriété "transform:"

  • scale(): la valeur par défaut de scale() est de 1.

Rappel: l'unité de mesure à utiliser pour les déplacements est le rem.
